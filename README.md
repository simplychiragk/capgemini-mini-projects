# Capgemini Mini Projects - Python

This Repository contains three Python mini-projects built using modular architecture and menu-driven CLI systems.

---

## 📂 Projects

### 💳 ATM Simulation
A simple ATM system that allows users to:
- Check balance  
- Deposit money  
- Withdraw money (with validation)  
- Maintain consistent state across modules  

---

### 🎮 Stone Paper Scissors Game
A CLI-based game where the user plays against the computer:
- Random computer choices  
- Win / Lose / Tie logic  
- Clean modular structure  
- Interactive user input  

---

### 📚 Library Management System
A basic library system using dictionaries for structured data:
- Add books  
- Issue books with student name, date, and duration  
- Return books with fine calculation  
- Track availability and user interactions
